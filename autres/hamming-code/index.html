Le code Hamming - Qu'est-ce?

Le code Hamming est, dans sa forme primaire, un algorithme qui permet l'autocorrection d'erreurs produits lors de l'envoi d'informations. Une forme avancée de ce code est utilisé dans les CD et DVD, qui ont besoin de ce genre d'algorithme, car chaque petite égratinure peut modifier les informations engravés dans l'appareil.
