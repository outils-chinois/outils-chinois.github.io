Grammaire

Allset Learning est un site permettant d'apprendre des structures grammaticales chinoises de niveau A1 à C1. Il suffit simplement de noter un caractère présent dans une structure grammaticale spéciale dans la barre de recherche, ou de simplement regarder les articles présents sur la page principale.

Feuille de Travail

Chinese Worksheet Generator est un site permettant de générer des feuilles de caractères chinois, figurant leur ordre d'écriture étape-par-étape. Cet outil a aussi beaucoup d'options de personnalisation, ce qui peut être outil pour imprimer la feuille résultante.

Lecture automatique (TTS)

TTS MP3 est un site qui permet de lire à voix haute un texte (à l'occurence en chinois) avec une prononciation adaptée. Il suffit tout simplement de copier-coller le texte dans le champ d'écriture et de lancer l'écoute pour faire fonctionner l'outil.

Dictionnaire

Pleco est une application mobile qui fait office d'un dictionnaire chinois. Il suffit simplement d'écrire le ou les caractères, le pinyin de ces caractères, leur traduction, ou simplement de les tracer sois-même pour retrouver toutes les informations les concernant.
